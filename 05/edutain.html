Shohini Ghose - A beginner's guide to quantum computing

  • Quantum computers work by controlling atoms and fundimental particles
  • A regular computer simulates heads and tails as a bit, a quantum computer assigns a fluid identity to the bit, a number between 0 and 1.
  • Quantum computers harness superposition and uncertainty in order to do a number of tasks including
    • Creation of private keys for encryption
    • Allow for exact calculations of particles and atoms in molecules for pharmaceutical use
    • Teleportation of data

Vikram Sharma - How quantum physics can make encryption stronger

  • Encryption uses keys to secure data. The key is generated many different ways
  • Software generates sudo random numbers that have a subtle pattern. Quantum keys are truely random so they can generate truely random numbers
  • Using Quantum Key Distribution utilizes a property of quantum particles that changes the private key if it is seen.
