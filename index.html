Well Rounded Technical Experience

From web development, to data science, to building and troubleshooting hardware, I have focused my education on learning from many computer science disciplines. Combined with a deeper understanding of mathematics, I have focused much of my career on understanding the statistical design behind machine learning algorithms. Other areas of focus include: Linux console, cyber security vulnerabilities, C-based programing, and Python

Hardware

As a child, hardware fascinated me as I watched my friends take apart and reassemble computers in front of me. In adulthood, I have found that this child-like wonder turned me into a hobbyist. Building computers for family and friends has given me the skills to debug and test individual hardware components

Machine Learning with Python

Using machine learning algorithms in python, we can create regression and categorical models to compute mathematics that we could not otherwise. Using these tools, I have created algorithms to predict enegrgy stock prices, and the the multivariate poverty index based on large datasets from kaggle
